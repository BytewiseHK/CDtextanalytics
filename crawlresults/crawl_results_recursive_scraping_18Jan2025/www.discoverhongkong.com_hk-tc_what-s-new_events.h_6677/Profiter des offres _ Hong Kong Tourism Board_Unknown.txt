Title: Profiter des offres | Hong Kong Tourism Board
Date: None
URL: https://www.discoverhongkong.com/fr/deals/in-town-offers/how-to-redeem.html

S'inscrire avec un numéro de portable
Entrez le code de vérification envoyé à votre téléphone mobile
Entrez vos informations et créez un mot de passe pour configurer votre compte
Visitez notre plate-forme numérique et choisissez les offres de votre choix
Cliquez sur « Ajouter à Mes Offres » pour recevoir les offres sélectionnées
Voir les offres sélectionnées dans « MES OFFRES » avant d'en profiter
Cliquez sur « utiliser le coupon » sur la page de l'offre sélectionnée
Scannez le QR code HKTB fourni par le personnel
Présentez le code-barres dans la confirmation au personnel (le cas échéant)
Cliquez sur « Confirmer » pour afficher le code promotionnel
Cliquez sur « Copier le code » et « Réservez maintenant » pour saisir le code sur le site Web du marchand au moment du paiement
